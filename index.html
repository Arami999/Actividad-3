Tema 2. Define los elementos que componen la estructuración de una página web:

1. Contenedor (wrapper)

es un elemento HTML que se utiliza para agrupar otros elementos y aplicarles estilos o comportamientos específicos.

2. Cabecera (header)

se utiliza para definir una sección de cabecera para un documento o una sección de un documento.

3. Contenido (content)

La propiedad content en CSS reemplaza el contenido con un valor generado. En HTML, el atributo content se utiliza para mostrar o contener el valor del nombre o del atributo HTTP-equit.

4. Menú (menu)

se utiliza para definir un conjunto de comandos o opciones que un usuario puede realizar.

5. Pie (footer)

se utiliza para definir el pie de página de un documento o de una sección dentro de un documento

6. Lateral (sidebar)

se utiliza para mostrar contenido complementario, como menús de navegación, widgets, anuncios, enlaces relacionados, o información adicional.

7. Etiqueta div

se utiliza como un contenedor genérico para agrupar y organizar otros elementos HTML.
